JavaScript API Playground

외부 CP사 또는 내부 웹서비스 기반 페이지에서 CDN 서버를 통해 사용하는 JavaScript API를 테스트할 수 있는 서비스입니다.

공통 설정

launcher > run

웹 페이지에서 PC클라이언트 (이하 런처)를 실행시키기 위한 메서드입니다.
조건에 따라 내부 프로세스에 점검 체크, 화이트 유저 체크를 추가로 진행합니다.

launcher > download

런처가 사용자 PC에 설치되어 있지 않는 경우 다운로드를 할 수 있는 메서드입니다.
download 메서드 호출 후 60초 이내에 재호출하면 메모리에 캐싱된 URL을 반환,
60초 초과 또는 두번째 인자 isSkipDownload를 true로 설정하면 api를 재조회해서 업데이트된 URL을 내려받습니다.

stoveTerms > checkAgreeState

약관 동의 상태를 조회합니다.

STC_REWE : (웹) 스토브 회원가입 시 사용
STC_REMO : (모바일 SDK) 스토브 회원가입 시 사용
SVC_AG : (PC 게임) 서비스 약관 동의 시 사용

maintenance > checkState

점검 진행 중인지 상태를 조회합니다.

sim > createCertState

임시 인증 상태 코드값을 조회합니다.

gameChecker > retrieveWebCheck

게임 실행에 필요한 정보를 조회합니다.

gameInfo > getGameId

게임 아이디를 조회합니다.
