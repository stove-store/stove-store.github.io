Stove PC SDK v3.0 .NET

소개

  • 스토브 플랫폼은 게임 배포, 업데이트, 지표 관리를 포함한, 게임 출시의 모든 과정을 쉽게 처리할 수 있는 통합 서비스를 제공합니다. 스토브 PC SDK를 사용하면 스토브 플랫폼이 제공하는 서비스를 게임에 쉽게 통합할 수 있습니다.

SDK종류

  • BaseSDK : 스토브 플랫폼을 사용하기 위한 모든 기본적인 정보를 관리하는 모듈입니다 (필수).
  • GameSupportSDK : 게임 지원 서비스를 연동할 수 있는 모듈입니다.
  • GamingServicesSDK : 게임 서비스를 연동할 수 있는 모듈입니다.
  • IAPSDK : 빌링을 연동할 수 있는 모듈입니다.
  • OwnershipSDK : 게임의 소유권을 확인할 수 있는 모듈입니다.
  • ViewSDK : 팝업창을 연동할 수 있는 모듈입니다.
  • PCBangSDK : PC방 기능을 연동할 수 있는 모듈입니다.

(.NET for C#, Unity)

  • BaseSDK_NET : BaseSDK에 대한 관리형 플러그인 입니다.
  • GameSupportSDK_NET : GameSupportSDK에 대한 관리형 플러그인 입니다.
  • GamingServicesSDK_NET : GamingServicesSDK에 대한 관리형 플러그인 입니다.
  • IAPSDK_NET : IAPSDK에 대한 관리형 플러그인 입니다.
  • OwnershipSDK_NET : OwnershipSDK에 대한 관리형 플러그인 입니다.
  • ViewSDK_NET : ViewSDK 대한 관리형 플러그인 입니다.
  • PCBangSDK_NET : PCBangSDK에 대한 관리형 플러그인 입니다.

  • IAPSDK의 오류 코드 페이지 IAPSDK_NET\Deploy\Misc\Enumerations.cs

(주의) C#, Unity 등 환경에서 Native SDK와 Managed Plugin은 쌍을 이루어 Import 해야 합니다. 즉, Base 모듈을 연동한다면 BaseSDK.dll와 BaseSDK_NET.dll 모두 Import가 필요합니다. View와 IAP 모듈의 경우 웹뷰 기능을 위해 제공하는 WebView2Loader.dll이 추가로 Import 되어야 합니다.
