Stove PC SDK v3.0 .NET

Introduction

  • Stove Platform provides an integrated service that can easily handle all processes of game release, including game distribution, updates, and metric management. By using the Stove PC SDK, you can easily integrate the services provided by the Stove Platform into your game.
  • The Stove PC SDK provides a variety of services that can be integrated into your game.

SDKs

  • BaseSDK : A module that manages all basic information needed to use the Stove Platform (required).
  • GameSupportSDK : A module that can be integrated with game support services.
  • GamingServicesSDK : A module that can be integrated with game services.
  • IAPSDK : A module that can be integrated with billing services.
  • OwnershipSDK : A module that can be used to check the ownership of a game.
  • ViewSDK : A module that can be integrated with popup windows.
  • PCBangSDK : A module that can be integrated with PC bang features.

Plugins(.NET for C#, Unity)

  • BaseSDK_NET : A managed plugin for BaseSDK.
  • GameSupportSDK_NET : A managed plugin for GameSupportSDK.
  • GamingServicesSDK_NET : A managed plugin for GamingServicesSDK.
  • IAPSDK_NET : A managed plugin for IAPSDK.
  • OwnershipSDK_NET : A managed plugin for OwnershipSDK.
  • ViewSDK_NET : A managed plugin for ViewSDK.
  • PCBangSDK_NET : A managed plugin for PCBangSDK.
